Password Reset / Windows unlock

Network üìÖ Last Updated: November 19, 2025

This guide covers resetting forgotten Windows passwords and unlocking locked accounts.

Overview

This article provides step-by-step instructions for resetting your Windows password if you've forgotten it, and for unlocking your account if it has been locked due to multiple failed login attempts.

üí° Tip: For security reasons, password resets may require administrator assistance in a network environment.

Resetting Your Windows Password

1 Boot from Installation Media

Insert your Windows installation USB or DVD and restart your computer. Press the appropriate key (usually F2, F10, F12, or Del) to enter BIOS/UEFI settings and change the boot order to boot from the USB/DVD first.

2 Access Recovery Options

On the installation screen, select your language preferences, then click "Repair your computer" in the bottom left corner.

3 Open Command Prompt

Navigate to Troubleshoot > Advanced options > Command Prompt. This will open an elevated command prompt.

4 Replace Utilman.exe

In the command prompt, type the following commands:

  • copy c:\windows\system32\utilman.exe c:\
  • copy c:\windows\system32\cmd.exe c:\windows\system32\utilman.exe
5 Restart and Access Command Prompt

Restart your computer normally. On the login screen, click the Ease of Access icon (bottom right) to open the command prompt.

6 Reset Password

In the command prompt, type:

  • net user [username] [newpassword]

Replace [username] with your actual username and [newpassword] with your desired password.

7 Restore Utilman.exe

Copy back the original file:

  • copy c:\utilman.exe c:\windows\system32\utilman.exe

Unlocking a Locked Windows Account

1 Wait for Lockout Period

Most accounts lock for a short period (usually 30 minutes) after failed attempts. Simply wait and try logging in again.

2 Contact Administrator

If you're on a network or domain, contact your IT support team to unlock the account. They can unlock it from Active Directory.

3 Use Alternative Login

If available, log in with an administrator account and unlock the user account using Computer Management.

Additional Information

If you're unable to reset your password using these methods, or if your computer is part of a corporate network, please contact the Service Desk for assistance.

Common Issues

  • BitLocker Encryption: If your drive is encrypted, you'll need the recovery key first
  • Domain Accounts: Password resets must be done through Active Directory by an administrator
  • Microsoft Account: For Microsoft accounts, use the online recovery options

Prerequisites

  1. Physical access to the computer
  2. Windows installation media (USB/DVD)
  3. Administrator privileges (for unlocking other accounts)

Summary

Following these steps should allow you to reset your forgotten Windows password or unlock a locked account. Remember to create strong, memorable passwords and enable additional security features like two-factor authentication when possible.
